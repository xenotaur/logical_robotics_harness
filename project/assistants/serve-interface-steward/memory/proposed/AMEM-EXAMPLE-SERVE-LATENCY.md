# AMEM-EXAMPLE-SERVE-LATENCY (example, proposed)

**Proposed lesson (illustrative only):** When the triage view is rendered from a
large control-plane state, prefer deriving a compact summary before rendering
detail, because operators reported the full render felt slow to scan.

**Why it matters:** Guides future interface work toward
`evidence_over_summary` and `accessibility_over_visual_minimalism` without
re-deriving the observation each time.

**Status:** Proposed. This is a worked example demonstrating the memory item
shape; it is not an accepted lesson. Promotion to `accepted/` requires a human
reviewer, a review timestamp, retained provenance (`evidence:` populated), and
validation. The assistant cannot accept its own memory.
