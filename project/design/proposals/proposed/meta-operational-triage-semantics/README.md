# Meta operational triage semantics proposal set

This proposal set defines semantics for the LRH Meta Operational Triage
Dashboard as an action-oriented, evidence-backed portfolio triage surface.

## Status

`proposed` / `not_started`

Documentation/control-plane proposal only. No runtime dashboard semantics,
classifier, routing, or UI behavior is implemented in this PR.
