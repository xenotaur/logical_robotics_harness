# Phase 4 — MCP and Agent Integration

This phase allows LRH to become a full agent harness rather than only a control-model tool.

## Deliverables

- MCP client integration
- tool registry
- agent backend abstraction
- guardrail evaluation engine
- approval boundaries

## Risks

- adding agentic complexity before the core harness is stable
- mixing tool protocol concerns with project control concerns
